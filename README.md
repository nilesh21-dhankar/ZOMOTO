# 🍽️ Restaurant Table Booking Website (Zomato-Inspired)

A *Zomato-inspired Restaurant Table Booking Website* developed using *HTML, CSS, and JavaScript*.  
This project demonstrates frontend design skills and basic booking flow logic.

---

## 🚀 Features

### 🏠 Restaurant Listing
- Displays restaurant details
- Clean and user-friendly UI
- Inspired by real-world food booking platforms

### 📅 Table Booking Form
- User can select date, time, and number of guests
- JavaScript-based input validation
- Smooth booking experience on frontend

### 📱 Responsive Design
- Works on mobile, tablet, and desktop devices
- Modern layout with proper spacing and alignment

---

## 🛠️ Tech Stack

- *HTML5* – Structure
- *CSS3* – Styling & responsiveness
- *JavaScript* – Booking logic & validation

---
